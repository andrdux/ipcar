Server requirements:
	-	PHP 5 or higher
	-	MySQL 5.3 or higher



Installation steps:

1) Execute MYSQL database scripts.

	a)	For english version execute /database/eng/structure/database.sql
		For russian version execute /database/rus/structure/database.sql		

	b)	For english version execute all scripts from folder /database/eng/data
		For russian version execute all scripts from folder /database/rus/data

   Database with name "ipcar" should be created


2) Update configuration in /sources/init.php file


	a) For english version: $configuration->lang = "eng";  
	   For russian version: $configuration->lang = "rus";  

	b) Update name and url of the website:
	   $configuration->wwwroot = 'http://www.siteroot.com';
	   $configuration->siteName = 'siteroot.com';

	c) Update email addresses:
	   $configuration->supportEmail = 'support@siteroot.com';
	   $configuration->contactEmail = 'contact@siteroot.com';

	d) Configure connection to the database:
	    $db = array(
	    "type" => 'mysql',
	    "host" => 'localhost',
	    "port" => "",
	    "name" => 'dbname',
	    "user" => 'dbuser',
	    "pass" => 'dbpassword'
	    );


Access management console:

1) Go to <web site root>/management

2) Login as administrator
	Email: admin@admin.com
	Password: 1
